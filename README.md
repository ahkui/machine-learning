# machine-learning

## Install OpenCV

| Environment Variable | Value |
| -------------------- | ----- |
| ENABLE_GPU           | false |
| OPENCV_VERSION       | 4.1.0 |

```shell
curl -L https://kui.im/install-opencv -o install-opencv
chmod +x install-opencv
./install-opencv
```
